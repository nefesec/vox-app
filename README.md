# Vox

Application de chat vocal & texte — comme Discord mais privé.

## Télécharger

👉 **[Voir les releases](https://github.com/nefesec/vox-app/releases/latest)**

| Plateforme | Fichier |
|---|---|
| Windows | `Vox 1.3.0.exe` — portable, aucune installation |
| Linux | `Vox-1.3.0.AppImage` |
| Android | `vox-1.3.0.apk` |

## Fonctionnalités

- Chat texte global & messages privés (DMs)
- Vocal WebRTC (peer-to-peer)
- Serveurs personnels (système d'invitation par code)
- Chiffrement AES-256-GCM des messages
- Panneau admin (ban, rôles, gestion serveurs)
